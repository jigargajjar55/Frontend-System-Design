# Frontend System Design

A comprehensive collection of frontend system design problems, JavaScript concepts, and machine coding challenges. This repository serves as a learning resource for mastering frontend architecture, design patterns, and low-level implementation details.

## üìÅ Repository Structure

### CodingQuestions/
Frontend coding challenges focusing on practical JavaScript problems:
- **Debounce_Throttling/** - Implementing debouncing and throttling techniques for performance optimization
- **FlattenObject/** - Flattening nested object structures
- **FolderLikeStructure/** - Building folder-like hierarchical data structures
- **FunctionCurrying/** - Understanding and implementing function currying
- **InfiniteScrolling/** - Implementing infinite scroll functionality
- **Layouts/** - CSS layout implementations and responsive design

### JSConcepts/
Core JavaScript concepts and their implementations:
- **BrowserStorage/** - Working with localStorage and sessionStorage
- **Bubbling_Trickling/** - Event propagation (bubbling and capturing)
- **CallbackFunctions/** - Callback patterns and async operations
- **Closures/** - Understanding lexical scope and closures
- **FirstClassFunctions/** - First-class function concepts
- **Hoisting/** - Variable and function hoisting behavior
- **Pollyfills/** - Custom implementations of built-in methods
  - ApplyPollyfill.js - Custom implementation of Function.prototype.apply
  - BindPollyfill.js - Custom implementation of Function.prototype.bind
  - CallPollyfill.js - Custom implementation of Function.prototype.call
  - EveryPollyfill.js - Custom implementation of Array.prototype.every
  - FilterPollyfill.js - Custom implementation of Array.prototype.filter
  - MapPollyfill.js - Custom implementation of Array.prototype.map
  - ReducePollyfill.js - Custom implementation of Array.prototype.reduce
- **Promises/** - Promise patterns and async handling
- **This/** - Understanding the `this` keyword in JavaScript

### MachineCoding/
Full-featured system design implementations:
- **AnalogClock/** - Building an analog clock using Canvas/SVG
- **MSTeamChatWindow/** - Building a chat window UI similar to Microsoft Teams
  - Organized CSS with base, components, layout, pages, theme, and utility folders
  - Modular JavaScript architecture
- **RatInMazeProblem/** - Solving the rat in maze pathfinding problem with visualization
- **SearchPagination/** - Implementing search with pagination
